## 页面三：欢迎页（路由：/welcome）

### 布局示意

```
┌──────────────────────────────────────────┐
│                                          │
│  ┌────────────────────────────────┐  ┌──┐│
│  │  web_oa 工作时间管理系统       │  │退出││    ← 标题 + 右上角登出按钮（danger 红色）
│  └────────────────────────────────┘  │登录││
│                                      └──┘│
│                                          │
│                                          │
│                                          │
│             ┌────────────────┐            │
│             │                │            │
│             │  hello zhangsan│            │    ← 居中大号字体显示，直接展示
│             │                │            │
│             └────────────────┘            │
│                                          │
│                                          │
│                                          │
│                                          │
└──────────────────────────────────────────┘
```

### 交互标注

| 元素 | 交互行为 |
|------|---------|
| 页面主体区域 | 页面挂载 (mounted) 时自动调用 GET /api/user/me（携带 Authorization: Bearer {token}），获取 username 后直接展示 hello {username}，无需用户点击 |
| 退出登录按钮（danger 红色） | 点击 → POST /api/auth/logout（携带 token）→ 后端将 jti 加入 jwt_blacklist → 前端清除 localStorage 的 token → 跳转 /login。即使接口调用失败，前端仍清 token 并跳转 |
| 标题区域 | 显示"web_oa 工作时间管理系统"，与登出按钮在同一行 |

### 无 token 被拦场景

```
用户直接键入 /welcome 但不带 token:

  ┌──────────────────────────────────────────┐
  │                                          │
  │   路由守卫 beforeEach 检测到              │
  │   localStorage 中无 token                │
  │                                          │
  │        ⬇                                 │
  │                                          │
  │   自动跳转至 /login                       │
  │                                          │
  └──────────────────────────────────────────┘
```

## 三页切换流程总览

```text
                    ┌───────────┐
                    │  /login   │
                    │  登录页   │
                    └─────┬─────┘
                          │
              ┌───────────┼───────────┐
              │           │           │
              ▼           ▼           │
        ┌──────────┐ ┌──────────┐     │
        │ 点击注册  │ │ 登录成功  │     │
        │ 跳转注册页│ │ 存 token │     │
        └────┬─────┘ │ 跳转欢迎页│     │
             │       └─────┬─────┘     │
             ▼             ▼           │
        ┌──────────┐ ┌──────────┐     │
        │ /register│ │ /welcome │     │
        │ 注册页   │ │ 欢迎页   │     │
        └────┬─────┘ └─────┬─────┘     │
             │             │           │
             │ 注册成功    │ 点击登出  │
             │ 存 token    │ 清 token  │
             │ 自动跳转    │ 跳转登录页│
             └──────┬──────┘           │
                    │                  │
                    └──────────────────┘

  已登录用户访问 /login → 自动跳转 /welcome
  无 token 用户访问 /welcome → 自动跳转 /login
```
